The Black Box Problem

Deep neural networks achieve remarkable performance — but what happens inside?

We see the input and the output. The intermediate layers remain opaque.

Existing approaches at the time:

  • Visualizing filters (Zeiler & Fergus, 2014)
  • Transferability studies (Yosinski et al., 2014)
  • Relevance propagation (Bach et al., 2015)

What’s missing? A simple, general-purpose tool to quantify what each layer knows — without modifying the model.

What Is a Linear Classifier Probe?

A probe is a small linear classifier attached to a layer’s features — trained independently of the model.

Tip

Analogy: Probes are like thermometers placed at many locations in a system. They measure temperature without changing it.

Key properties:

  • Probes never influence the model (gradients are blocked via stop_gradient)
  • Since it’s a linear classifier with softmax cross-entropy → convex problem (no local minima)
  • Can be attached to every layer simultaneously

Mathematical Formulation

At every layer k, take features H_k and fit a linear classifier:

f_k: H_k \to [0,1]^D, \quad h_k \mapsto \text{softmax}(Wh_k + b)

  • (W, b) are learned to minimize cross-entropy — independently from the main model
  • \mathcal{L}_k^{\text{train}}, \mathcal{L}_k^{\text{valid}}, \mathcal{L}_k^{\text{test}} = probe loss at layer k on train/valid/test sets

Important

Convexity guarantee: Linear classifier + softmax cross-entropy is a convex problem. No local minima — we always find the global optimum.

The Information Theory Insight

The Data Processing Inequality states that for X \to Y \to Z:

I(X;Z) \leq I(X;Y)

Each layer can only lose information, never gain it.

Yet deeper layers are better for classification. How?

Resolution: Neural networks distill computationally useful representations, not maximal information.

Tip

A raw image file contains millions of bytes — pixel colors, noise, background details. That’s maximum information, but it’s useless for answering ‘is there a lion in this picture?’ The network’s job is to throw away the irrelevant stuff and keep only what matters for the task.

Observations: Even random (untrained) layers provide useful features — error drops from 8% to 2%. After training, error monotonically improves at every layer.

The Key Finding: Monotonic Separability

Linear separability of features increases monotonically with depth.

This is not part of the training objective — only the last layer is directly pressured by cross-entropy. Yet every intermediate layer progressively improves.

monotonicViz = {
  const W = 900, H = 360;
  const svg = d3.create("svg")
    .attr("viewBox",`0 0 ${W} ${H}`)
    .style("width","100%").style("max-width",`${W}px`)
    .style("font-family","system-ui, sans-serif");

  svg.append("rect").attr("width",W).attr("height",H)
    .attr("rx",10).attr("fill","#f8f9fa").attr("stroke","#dee2e6");

  // LEFT: scatter morphing
  const scW = W*0.42, padS = 40;

  function randn(){
    let u=0,v=0;
    while(!u) u=Math.random();
    while(!v) v=Math.random();
    return Math.sqrt(-2*Math.log(u))*Math.cos(2*Math.PI*v);
  }

  const Np = 90, seed = [];
  for(let i=0;i<Np;i++){
    const cls = i<Np/2 ? 0 : 1;
    const cx0 = cls===0 ? -0.3 : 0.3;
    const cy0 = cls===0 ? 0.2 : -0.2;
    seed.push({
      cls,
      x0: cx0 + randn()*1.3,
      y0: cy0 + randn()*1.3,
      x1: (cls===0 ? -2.5 : 2.5) + randn()*0.4,
      y1: (cls===0 ? -1.2 : 1.2) + randn()*0.4
    });
  }
  const xSc = d3.scaleLinear().domain([-4.5,4.5]).range([padS,scW-10]);
  const ySc = d3.scaleLinear().domain([-3.5,3.5]).range([H-30,50]);

  svg.append("text").attr("x",scW/2).attr("y",28)
    .attr("text-anchor","middle").attr("font-size",13).attr("fill","#343a40").attr("font-weight",700)
    .text("Feature space at current layer");

  const dots = svg.selectAll(".dot").data(seed).join("circle")
    .attr("r",3.5).attr("opacity",0.7);

  // separator line
  const sepLine = svg.append("line")
    .attr("stroke","#868e96").attr("stroke-width",1.5).attr("stroke-dasharray","6,4");

  // RIGHT: probe accuracy curve
  const rX0 = scW + 60, rW = W - rX0 - 40;
  const nLayers = 16;
  // simulated monotonic decreasing error (like ResNet-50)
  const errors = [0.99,0.94,0.89,0.88,0.87,0.82,0.79,0.76,0.77,0.69,0.67,0.62,0.57,0.51,0.41,0.31];
  const accs = errors.map(e => 1-e);

  const axX = d3.scaleLinear().domain([0,nLayers-1]).range([rX0, rX0+rW]);
  const axY = d3.scaleLinear().domain([0,0.75]).range([H-40, 55]);

  // axes
  svg.append("line").attr("x1",rX0).attr("y1",H-40).attr("x2",rX0+rW).attr("y2",H-40).attr("stroke","#adb5bd");
  svg.append("line").attr("x1",rX0).attr("y1",H-40).attr("x2",rX0).attr("y2",50).attr("stroke","#adb5bd");
  svg.append("text").attr("x",rX0+rW/2).attr("y",H-10)
    .attr("text-anchor","middle").attr("font-size",10).attr("fill","#495057").text("Layer depth →");
  svg.append("text").attr("x",rX0-8).attr("y",(55+H-40)/2)
    .attr("text-anchor","middle").attr("font-size",10).attr("fill","#495057")
    .attr("transform",`rotate(-90,${rX0-8},${(55+H-40)/2})`).text("Probe accuracy");

  svg.append("text").attr("x",rX0+rW/2).attr("y",28)
    .attr("text-anchor","middle").attr("font-size",13).attr("fill","#343a40").attr("font-weight",700)
    .text("Accuracy vs depth (ResNet-50 style)");

  // full curve (faded)
  const lineFn = d3.line().curve(d3.curveMonotoneX);
  const fullCurve = accs.map((a,i) => [axX(i), axY(a)]);
  svg.append("path").attr("d",lineFn(fullCurve))
    .attr("fill","none").attr("stroke","#dee2e6").attr("stroke-width",2);

  // animated highlight curve
  const highlightPath = svg.append("path")
    .attr("fill","none").attr("stroke","#2f9e44").attr("stroke-width",3);

  // animated dot on curve
  const curveDot = svg.append("circle").attr("r",6).attr("fill","#2f9e44").attr("opacity",0.9);
  // corresponding dots on scatter
  const sepX = xSc(0);

  let frame = 0;
  function anim(){
    frame++;
    const period = 400;
    const t = (frame % period) / period; // 0→1
    const layerIdx = Math.floor(t * (nLayers-1));
    const layerFrac = t;

    // scatter interpolation
    dots
      .attr("cx", d => xSc(d.x0*(1-layerFrac) + d.x1*layerFrac))
      .attr("cy", d => ySc(d.y0*(1-layerFrac) + d.y1*layerFrac))
      .attr("fill", d => d.cls===0 ? "#e03131" : "#1971c2");

    sepLine.attr("x1",sepX).attr("y1",50).attr("x2",sepX).attr("y2",H-30).attr("opacity",0.5);

    // curve highlight
    const partial = fullCurve.slice(0, layerIdx+2);
    highlightPath.attr("d",lineFn(partial));

    // dot
    curveDot.attr("cx",axX(layerIdx)).attr("cy",axY(accs[layerIdx]));

    requestAnimationFrame(anim);
  }
  anim();

  return svg.node();
}

Note

This “greedy” layerwise improvement suggests each layer refines the representation slightly, making it more linearly separable — an emergent property of standard backpropagation.

Practical Concerns

Train vs Validation Loss

  • \mathcal{L}_k^{\text{valid}} is more meaningful to monitor
  • But \mathcal{L}_k^{\text{train}} is easier to compute in all settings
  • For building intuition, both are acceptable

Reporting Metric

  • Paper reports classification error (not cross-entropy)
  • Ultimately the quantity practitioners care about
  • Top-5 error also possible

The probe reveals: The first 64 layers are completely dead — flat high error. The skip connection let the model bypass them entirely. Without probes, this failure would be invisible from the final loss alone.

Probing with Different Labels

Probes are not limited to the original task’s classes:

“Is there any information about factor _____ present in this part of the model?”

Examples:

  • Probe a vision model for color information
  • Probe for texture vs shape
  • Probe an NLP model for syntax or sentiment
  • Transfer across domains (same layers, new labels)

Multi-Layer Probes?

  • Risk of overfitting in the probe itself
