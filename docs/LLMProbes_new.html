The Black Box Problem

Deep neural networks achieve remarkable performance — but what happens inside?

We see the input and the output. The intermediate layers remain opaque.

What’s missing? A simple, general-purpose tool to quantify what each layer knows — without modifying the model.

Alain & Bengio solves this problem!

What Is a Linear Classifier Probe?

A probe is a small linear classifier attached to a layer’s features — trained independently of the model.

Analogy: Probes are like thermometers placed at many locations in a system. They measure temperature without changing it.

Key properties:

  • Probes never influence the model (gradients are blocked via stop_gradient)
  • Since it’s a linear classifier with softmax cross-entropy → convex problem (no local minima)
  • Can be attached to every layer simultaneously

Mathematical Formulation

At every layer k, take features H_k and fit a linear classifier:

f_k: H_k \to [0,1]^D, \quad h_k \mapsto \text{softmax}(Wh_k + b)

  • (W, b) are learned to minimize cross-entropy — independently from the main model
  • \mathcal{L}_k^{\text{train}}, \mathcal{L}_k^{\text{valid}}, \mathcal{L}_k^{\text{test}} = probe loss at layer k on train/valid/test sets

Convexity guarantee: Linear classifier + softmax cross-entropy is a convex problem. No local minima — we always find the global optimum.

The Information Theory Insight

The Data Processing Inequality states that for X \to Y \to Z:

I(X;Z) \leq I(X;Y)

Each layer can only lose information, never gain it.

Yet deeper layers are better for classification. How?

Resolution: Neural networks distill computationally useful representations, not maximal information.

A raw image file contains millions of bytes — pixel colors, noise, background details. That’s maximum information, but it’s useless for answering ‘is there a lion in this picture?’ The network’s job is to throw away the irrelevant stuff and keep only what matters for the task.

Model: 2 conv layers + 2 fully-connected layers (the standard TensorFlow MNIST example).

Probes inserted at every sub-operation: before/after convolution, activation, pooling.

The Key Finding: Monotonic Separability

Setup: Probes attached to all 16 residual layers of a pre-trained ResNet-50 on ImageNet (1000 classes, ~1.2M images).

Result: Probe validation error decreases almost perfectly monotonically — deeper features are always more linearly separable. This is not part of the training objective — only the last layer is directly pressured by cross-entropy.

Practical Concerns

Train vs Validation Loss

  • \mathcal{L}_k^{\text{valid}} is more meaningful to monitor
  • But \mathcal{L}_k^{\text{train}} is easier to compute in all settings
  • For building intuition, both are acceptable

Overfitting danger

  • With too many features, probes can memorize random labels.
  • Dimension reduction is essential for meaningful measurements.

From Vision Probes to LLM Probes

Alain & Bengio showed probes reveal structure in vision models. Can the same idea work for language models?

The debate on LLMs:

  • One view: LLMs learn a massive collection of surface correlations but lack any coherent understanding
  • Alternative: LLMs, through compressing data, learn coherent internal models of the process that generated the text — a world model

Gurnee & Tegmark’s approach: Ty to extract an actual map of the world and a timeline of history from inside an LLM’s hidden layers.

They use linear regression probes (extending Alain & Bengio’s classification probes) on the Llama-2 (7B–70B) and Pythia (160M–6.9B) model families.

Approach: Train linear regression probes on internal activations to predict real-world coordinates (latitude/longitude) or timestamps.

Method: Ridge Regression Probes

Key shift from Alain & Bengio: Instead of classification (softmax), use linear ridge regression to predict continuous targets.

\hat{W} = \arg\min_W \|Y - AW\|^2_2 + \lambda \|W\|^2_2 = (A^TA + \lambda I)^{-1}A^TY

where A \in \mathbb{R}^{n \times d_{model}} is the activation matrix and Y contains lat/lon or timestamps.

Process for each entity:

  1. Feed entity name through the LLM (e.g. “Los Angeles”)
  2. Extract hidden state at the last token for each layer
  3. This gives an n \times d_{model} activation dataset per layer
  4. Fit ridge regression to predict coordinates
  5. Evaluate on held-out test set

Evaluation metrics: R^2, Spearman rank correlation

The Period Token Effect (Headlines)

Surprising finding: For news headlines, probing on a period token appended after the headline gives better temporal predictions than probing on the last headline token itself.

What they tested:

  • empty: just the headline text
  • empty_wo_period: headline without period, last token
  • when_w_period: “When was this published?” + headline + “.”
  • when_wo_period: “When was this published?” + headline

What this suggests:

Period tokens act as summary positions — the model uses them to consolidate information about the preceding sentence. When the model hits “.”, it compresses the key facts (including temporal context) into that token’s representation.

This is consistent with how autoregressive models work — special punctuation tokens often serve as natural “checkpoint” positions where information gets aggregated.

    • But the probe needs extra parameters to convert from the model’s coordinate system to human coordinates (lat/lon, years)

The model has its own internal “coordinate system” for space and time — it’s just not aligned with our latitude/longitude grid.

        const yVal = yBase + randn() * n.noise;

          dotsForNeuron.push(dot);
        }
      }
    });

  });

    frame++;
    const period = 200;
    const t = (frame % period) / period;

    allScatterDots.forEach(dotsArr => {
      const visible = Math.floor(t * dotsArr.length);
      dotsArr.forEach((dot, i) => {
        dot.attr("opacity", i < visible ? 0.45 : 0);
      });
    });

  }

Key Takeaways — Gurnee & Tegmark

What they showed:

  • LLMs learn linear representations of space and time at multiple scales
  • These representations are robust to prompting and unified across entity types
  • Individual space/time neurons exist and causally influence predictions
  • Larger models + more training data → better representations

What this means:

LLMs don’t just learn surface correlations — they build structured internal models with genuine spatial and temporal geometry. These are basic ingredients of a world model.
